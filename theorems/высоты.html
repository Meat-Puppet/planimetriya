Теорема о высотах треугольника

hₐ = 2S / a = b·sin C = c·sin B

Формулировка

Высоты треугольника (или их продолжения) пересекаются в одной точке, называемой ортоцентром.

Обозначения

Пусть дан треугольник ABC. AA₁, BB₁, CC₁ — высоты, опущенные из вершин A, B, C на противоположные стороны (или их продолжения).

H — точка пересечения высот (ортоцентр).

a, b, c — длины сторон BC, AC, AB.

hₐ, h_b, h_c — длины высот к сторонам a, b, c.

S — площадь треугольника.

Формулы высот: hₐ = 2S / a = b·sin C = c·sin B

Доказательство (пересечение в одной точке)

Шаг 1. Проведём через вершины треугольника ABC прямые, параллельные противоположным сторонам. Они образуют новый треугольник A'B'C'.
Шаг 2. Четырёхугольники ABCB', ACBC', BACA' — параллелограммы, поэтому A'B = AC, A'C = AB и т.д.
Шаг 3. Вершины исходного треугольника A, B, C являются серединами сторон треугольника A'B'C'.
Шаг 4. Высоты треугольника ABC перпендикулярны его сторонам, а значит, перпендикулярны сторонам треугольника A'B'C'.
Шаг 5. Поскольку высоты проходят через середины сторон треугольника A'B'C' и перпендикулярны им, они являются серединными перпендикулярами треугольника A'B'C'.
Шаг 6. Серединные перпендикуляры любого треугольника пересекаются в одной точке. Следовательно, высоты треугольника ABC также пересекаются в одной точке.

Теорема доказана.

Формула высоты

Шаг 1. Площадь треугольника можно вычислить по формуле: S = ½·a·hₐ.
Шаг 2. Выразим высоту: hₐ = 2S / a.
Шаг 3. Площадь также равна: S = ½·b·c·sin A.
Шаг 4. Подставляя, получаем: hₐ = (b·c·sin A) / a.
Шаг 5. По теореме синусов: b / sin B = a / sin A ⇒ sin A = a·sin B / b.
Шаг 6. Подставляем: hₐ = (b·c·a·sin B) / (a·b) = c·sin B.
Шаг 7. Аналогично, hₐ = b·sin C.

Формулы выведены.

Следствия из теоремы

  • Ортоцентр — точка пересечения высот — может лежать внутри треугольника (остроугольный), на вершине прямого угла (прямоугольный) или вне треугольника (тупоугольный).
  • Высоты обратно пропорциональны сторонам: hₐ : h_b : h_c = 1/a : 1/b : 1/c.
  • В прямоугольном треугольнике высота, опущенная на гипотенузу, делит его на два подобных треугольника.
  • Расстояние от вершины до ортоцентра: AH = 2R·cos A, где R — радиус описанной окружности.
  • Ортоцентр, центроид и центр описанной окружности лежат на одной прямой — прямой Эйлера.
  • Основания высот образуют ортотреугольник, обладающий свойством минимального периметра среди вписанных треугольников.

Примеры применения

Пример 1. В треугольнике ABC сторона a = 10, площадь S = 30. Найдите высоту hₐ.

Решение: hₐ = 2S / a = 2·30 / 10 = 6.

Пример 2. В треугольнике сторона c = 8, угол B = 30°. Найдите высоту hₐ.

Решение: hₐ = c·sin B = 8·sin 30° = 8·0,5 = 4.

Пример 3. В треугольнике высоты равны 12, 15 и 20. Найдите стороны треугольника, если его площадь равна 60.

Решение: a = 2S / hₐ = 120 / 12 = 10; b = 120 / 15 = 8; c = 120 / 20 = 6. Треугольник со сторонами 6, 8, 10 — прямоугольный.

Историческая справка

Свойство высот пересекаться в одной точке было известно ещё древнегреческим математикам, но строгое доказательство впервые привёл Архимед в работе «О равновесии плоских фигур».

Леонард Эйлер в XVIII веке установил, что ортоцентр H, центроид G и центр описанной окружности O лежат на одной прямой (прямая Эйлера), причём HG = 2·GO.

Термин «ортоцентр» происходит от греческих слов «ортос» (прямой) и «кентрон» (центр).

Интересные факты

  • Ортоцентр — одна из четырёх замечательных точек треугольника (наряду с центроидом, инцентром и центром описанной окружности).
  • В остроугольном треугольнике ортоцентр является инцентром ортотреугольника (треугольника, образованного основаниями высот).
  • Если треугольник тупоугольный, две высоты падают на продолжения сторон, и ортоцентр лежит вне треугольника.
  • Точки, симметричные ортоцентру относительно сторон треугольника, лежат на описанной окружности.
  • Высоты используются в геодезии для определения высот местности и в архитектуре для расчёта уклонов крыш.
