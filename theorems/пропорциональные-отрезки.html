Пропорциональные отрезки в прямоугольном треугольнике

h² = a_c · b_c   |   a² = c · a_c   |   b² = c · b_c

Формулировка

В прямоугольном треугольнике высота, опущенная из вершины прямого угла на гипотенузу, есть среднее пропорциональное между проекциями катетов на гипотенузу. Каждый катет есть среднее пропорциональное между гипотенузой и его проекцией на гипотенузу.

Обозначения

Пусть дан прямоугольный треугольник ABC с прямым углом C.

a = BC — катет, противолежащий углу A.

b = AC — катет, противолежащий углу B.

c = AB — гипотенуза.

h = CD — высота, опущенная из прямого угла C на гипотенузу AB.

a_c = BD — проекция катета a на гипотенузу.

b_c = AD — проекция катета b на гипотенузу.

Тогда:

h² = a_c · b_c

a² = c · a_c

b² = c · b_c

Доказательство (через подобие)

Шаг 1. Проведём высоту CD из прямого угла C на гипотенузу AB. Получим два прямоугольных треугольника: ACD и BCD.
Шаг 2. Треугольник ACD подобен треугольнику ABC по двум углам (∠A — общий, ∠ADC = ∠ACB = 90°).
Шаг 3. Из подобия ACD ∾ ABC: AC/AB = AD/AC ⇒ b/c = b_c/b ⇒ b² = c · b_c.
Шаг 4. Треугольник BCD подобен треугольнику ABC по двум углам (∠B — общий, ∠BDC = ∠BCA = 90°).
Шаг 5. Из подобия BCD ∾ ABC: BC/AB = BD/BC ⇒ a/c = a_c/a ⇒ a² = c · a_c.
Шаг 6. Треугольники ACD и BCD подобны (оба подобны ABC, следовательно, подобны друг другу).
Шаг 7. Из подобия ACD ∾ BCD: AD/CD = CD/BD ⇒ b_c/h = h/a_c ⇒ h² = a_c · b_c.

Все три соотношения доказаны.

Следствия из теоремы

  • Высота h = √(a_c · b_c).
  • Катет a = √(c · a_c).
  • Катет b = √(c · b_c).
  • Сложив a² + b² = c·a_c + c·b_c = c(a_c + b_c) = c² — получаем доказательство теоремы Пифагора.
  • Отношение проекций катетов: a_c / b_c = a² / b².
  • Площадь прямоугольного треугольника: S = ½·a·b = ½·c·h.

Примеры применения

Пример 1. В прямоугольном треугольнике катеты равны 6 и 8. Найдите высоту, опущенную на гипотенузу.

Решение: c = √(6² + 8²) = 10. Проекции: a_c = a²/c = 36/10 = 3,6; b_c = b²/c = 64/10 = 6,4. h = √(3,6 · 6,4) = √23,04 = 4,8.

Пример 2. Высота прямоугольного треугольника делит гипотенузу на отрезки 4 и 9. Найдите катеты.

Решение: c = 4 + 9 = 13. a = √(13 · 9) = √117 = 3√13; b = √(13 · 4) = √52 = 2√13.

Пример 3. В прямоугольном треугольнике проекция одного катета на гипотенузу равна 5, а высота равна 12. Найдите гипотенузу.

Решение: h² = a_c · b_c ⇒ 144 = 5 · b_c ⇒ b_c = 28,8. c = 5 + 28,8 = 33,8.

Историческая справка

Теорема о пропорциональных отрезках в прямоугольном треугольнике была известна ещё в Древней Греции. Евклид в «Началах» (Книга VI, Предложение 8) формулирует её как свойство подобных треугольников.

Эти соотношения лежат в основе доказательства теоремы Пифагора через подобие, которое считается одним из самых элегантных. В средние века теорему использовали для геометрических построений и вычислений.

В современной геометрии пропорциональные отрезки — важный инструмент для решения задач с прямоугольными треугольниками.

Интересные факты

  • Высота к гипотенузе делит прямоугольный треугольник на два треугольника, подобных исходному и друг другу.
  • Теорема о пропорциональных отрезках — это по сути геометрическая интерпретация среднего геометрического.
  • С помощью этих соотношений можно построить отрезок, равный √ab, используя только циркуль и линейку.
  • В физике аналогичные соотношения возникают при расчёте электрических цепей с параллельным соединением.
  • Теорема Эвклида о высоте (h² = a_c·b_c) иногда называется «теоремой о высоте прямоугольного треугольника».
