Формулы планиметрии

Все основные формулы и соотношения из теорем планиметрии

📐 Треугольники (общие)

Сумма углов
∠A + ∠B + ∠C = 180°
Внешний угол
∠внеш = ∠A + ∠B
Теорема синусов
a/sin A = b/sin B = c/sin C = 2R
Теорема косинусов
c² = a² + b² − 2ab·cos C
Теорема тангенсов
(a−b)/(a+b) = tg[(A−B)/2] / tg[(A+B)/2]
Теорема Стюарта
b²m + c²n = a(d² + mn)
Свойство биссектрисы
BD/DC = AB/AC
Длина биссектрисы
lₐ = √(bc − mn)
Медиана
mₐ = ½√(2b² + 2c² − a²)
Свойство медиан
AO:OM = 2:1
Высота
hₐ = 2S/a = b·sin C
Серединный перпендикуляр
MA = MB
Неравенство треугольника
a + b > c
Формула Герона
S = √(p(p−a)(p−b)(p−c)), p = (a+b+c)/2
Теорема Эйлера (треугольник)
d² = R(R − 2r)

🔺 Равнобедренный треугольник

Углы при основании
∠A = ∠B
Признак равнобедренности
∠A = ∠B ⇒ a = b
Теорема Штейнера—Лемуса
lₐ = l_b ⇒ a = b

🔷 Равенство треугольников

I признак (две стороны и угол)
AB=A₁B₁, AC=A₁C₁, ∠A=∠A₁ ⇒ ∆=∆
II признак (сторона и два угла)
AB=A₁B₁, ∠A=∠A₁, ∠B=∠B₁ ⇒ ∆=∆
III признак (три стороны)
AB=A₁B₁, BC=B₁C₁, AC=A₁C₁ ⇒ ∆=∆

📏 Прямоугольный треугольник

Теорема Пифагора
c² = a² + b²
Высота к гипотенузе
h² = a_c · b_c
Катет через проекцию
a² = c · a_c
Катет против 30°
a = c/2
Медиана к гипотенузе
m = c/2 = R

📐 Подобие треугольников

I признак (по двум углам)
∠A = ∠A₁, ∠B = ∠B₁ ⇒ ∆ABC ∾ ∆A₁B₁C₁
II признак (две стороны и угол)
a/a₁ = b/b₁, ∠C = ∠C₁ ⇒ ∆ABC ∾ ∆A₁B₁C₁
III признак (три стороны)
a/a₁ = b/b₁ = c/c₁ ⇒ ∆ABC ∾ ∆A₁B₁C₁
Теорема Фалеса
AD/DB = AE/EC
Средняя линия
MN ∥ AC, MN = ½ AC

🔲 Четырёхугольники

Сумма углов n-угольника
Σ = (n − 2) · 180°
Средняя линия трапеции
m = (a + b) / 2
Площадь трапеции
S = (a + b) · h / 2
Параллелограмм (стороны)
AB = CD, BC = AD, AB ∥ CD, BC ∥ AD
Параллелограмм (диагонали)
AO = OC, BO = OD
Площадь параллелограмма
S = a · hₐ = ab · sin α
Ромб (диагонали)
d₁ ⟂ d₂, S = ½ d₁ d₂
Прямоугольник (диагонали)
d₁ = d₂ = √(a² + b²)
Квадрат
S = a², d = a√2
Теорема Вариньона
Середины сторон → параллелограмм
Теорема Птолемея
AC · BD = AB · CD + BC · AD
Вписанный четырёхугольник
∠A + ∠C = ∠B + ∠D = 180°
Формула Брахмагупты
S = √((p−a)(p−b)(p−c)(p−d)), p = (a+b+c+d)/2
Описанный четырёхугольник
AB + CD = BC + AD

⚪ Окружность

Вписанный угол
∠ABC = ½ ∪AC
Центральный угол
∠AOB = ∪AB
Угол между касательной и хордой
∠ = ½ ∪ (заключённой дуги)
Касательная и секущая
AB² = AC · AD
Пересекающиеся хорды
AE · EB = CE · ED
Отрезки касательных
AB = AC
Длина окружности
C = 2πR
Площадь круга
S = πR²
Длина дуги
L = πR · α / 180°
Площадь сектора
S = πR² · α / 360°

📍 Замечательные точки и линии

Центроид (медианы)
AG : GM = 2 : 1
Инцентр (биссектрисы)
r = S / p
Центр описанной окружности
R = abc / (4S)
Ортоцентр
AH = 2R·cos A
Прямая Эйлера
OH = 3·OG, HG = 2·GO
Окружность девяти точек
R₉ = R / 2
Точка Жергонна
Пересечение чевиан к точкам касания вписанной
Точка Нагеля
Пересечение чевиан к точкам касания вневписанных
