Катет, лежащий против угла 30°

a = c / 2

Формулировка

В прямоугольном треугольнике катет, лежащий против угла 30°, равен половине гипотенузы.

Обратно: если в прямоугольном треугольнике катет равен половине гипотенузы, то угол, лежащий против этого катета, равен 30°.

Обозначения

Пусть дан прямоугольный треугольник ABC с прямым углом C.

∠A = 30° — острый угол.

a = BC — катет, противолежащий углу A (углу 30°).

b = AC — катет, прилежащий к углу A.

c = AB — гипотенуза.

Тогда: a = c / 2

Доказательство (прямое)

Шаг 1. Рассмотрим прямоугольный треугольник ABC с ∠A = 30° и прямым углом C.
Шаг 2. Пристроим к треугольнику ABC равный ему треугольник ABD так, чтобы катет AC был общим, а точки B и D лежали по разные стороны от AC.
Шаг 3. Тогда ∠BAD = ∠BAC + ∠CAD = 30° + 30° = 60°.
Шаг 4. Треугольник ABD — равнобедренный (AB = AD). Угол при вершине A равен 60°, следовательно, треугольник ABD равносторонний.
Шаг 5. В равностороннем треугольнике все стороны равны: AB = BD = AD.
Шаг 6. BC — высота (и медиана) в равностороннем треугольнике ABD, поэтому BC = BD / 2.
Шаг 7. Так как BD = AB = c, получаем a = BC = c / 2.

Прямое утверждение доказано.

Доказательство (обратное)

Шаг 1. Пусть в прямоугольном треугольнике ABC катет a равен половине гипотенузы c: a = c/2.
Шаг 2. По определению синуса: sin A = a / c = (c/2) / c = 1/2.
Шаг 3. Угол A — острый, и sin A = 1/2. Из тригонометрии известно, что единственный острый угол с таким синусом — 30°.
Шаг 4. Следовательно, ∠A = 30°.

Обратное утверждение доказано.

Следствия из теоремы

  • В прямоугольном треугольнике с углом 30° второй острый угол равен 60°.
  • Катет, прилежащий к углу 30°, равен c·cos 30° = c√3/2.
  • Отношение катетов в таком треугольнике: a : b = 1 : √3.
  • Площадь треугольника: S = ½·a·b = c²√3 / 8.
  • Если в прямоугольном треугольнике один из катетов в два раза короче гипотенузы, то противолежащий угол равен 30°.

Примеры применения

Пример 1. В прямоугольном треугольнике гипотенуза равна 10, а один из углов 30°. Найдите катет, лежащий против этого угла.

Решение: a = c / 2 = 10 / 2 = 5.

Пример 2. В прямоугольном треугольнике катет равен 6 и равен половине гипотенузы. Найдите углы треугольника.

Решение: c = 2·6 = 12. Угол против катета: sin α = 6/12 = 0,5 ⇒ α = 30°. Второй острый угол = 60°.

Пример 3. В прямоугольном треугольнике с углом 30° гипотенуза равна 20. Найдите второй катет.

Решение: Катет против 30°: a = 20/2 = 10. Второй катет: b = √(20² − 10²) = √(400 − 100) = √300 = 10√3.

Историческая справка

Свойство катета, лежащего против угла 30°, было известно ещё в Древнем Египте и Вавилоне. Оно непосредственно следует из свойств равностороннего треугольника.

Евклид в «Началах» доказывает это свойство через построение равностороннего треугольника. В средневековой Европе теорема широко использовалась в архитектуре и строительстве для построения прямых углов и расчёта уклонов.

В современной тригонометрии это свойство является частным случаем определения синуса: sin 30° = 1/2.

Интересные факты

  • Треугольник с углами 30°, 60°, 90° — один из двух «специальных» прямоугольных треугольников в тригонометрии (второй — 45°, 45°, 90°).
  • Стороны такого треугольника относятся как 1 : √3 : 2.
  • Это свойство часто используется в задачах без применения тригонометрических функций.
  • В черчении угольник с углами 30° и 60° — один из основных инструментов.
  • Свойство катета против 30° позволяет легко находить высоту равностороннего треугольника: h = a√3/2.
