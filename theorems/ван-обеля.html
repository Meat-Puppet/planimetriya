Теорема Ван-Обеля

AF/FB + AE/EC = AD/OD

Формулировка

Пусть в треугольнике ABC чевианы AD, BE, CF пересекаются в одной точке O. Тогда для любой из чевиан (например, AD) выполняется соотношение:

AF/FB + AE/EC = AD/OD

Аналогичные равенства справедливы и для других чевиан.

Обозначения

Пусть дан треугольник ABC.

AD, BE, CF — чевианы, пересекающиеся в точке O.

D лежит на BC, E на CA, F на AB.

Тогда: AF/FB + AE/EC = AD/OD

Аналогично: BF/FA + BD/DC = BE/OE и CD/DB + CE/EA = CF/OF

Доказательство

Шаг 1. Используем отношения площадей треугольников с общей высотой.
Шаг 2. AF/FB = S(AFC)/S(BFC) = S(AFO)/S(BFO) = S(AOC)/S(BOC).
Шаг 3. Аналогично, AE/EC = S(ABE)/S(CBE) = S(AEO)/S(CEO) = S(AOB)/S(BOC).
Шаг 4. Складываем: AF/FB + AE/EC = (S(AOC) + S(AOB)) / S(BOC).
Шаг 5. С другой стороны, AD/OD = S(ABD)/S(OBD) = S(ACD)/S(OCD) = (S(ABD) + S(ACD)) / (S(OBD) + S(OCD)) = S(ABC) / S(BOC).
Шаг 6. Но S(ABC) = S(AOB) + S(BOC) + S(AOC). Отсюда S(AOC) + S(AOB) = S(ABC) − S(BOC).
Шаг 7. Таким образом, AF/FB + AE/EC = (S(ABC) − S(BOC)) / S(BOC) = S(ABC)/S(BOC) − 1 = AD/OD − 1? Проверим: AD/OD = S(ABC)/S(BOC). Тогда AF/FB + AE/EC = AD/OD.

Теорема доказана.

Следствия из теоремы

  • Для центроида G (точка пересечения медиан): AF/FB = 1, AE/EC = 1, AD/GD = 3, формула даёт 1 + 1 = 3? AD/GD = 3, OD = GD, AD = 3·GD, AD/GD = 3. Верно.
  • Для инцентра I: отношения выражаются через стороны, теорема Ван-Обеля связывает их с положением инцентра на биссектрисе.
  • Теорема Ван-Обеля обобщает свойства многих замечательных точек треугольника.
  • С помощью теоремы можно находить отношения, в которых чевианы делятся точкой пересечения.

Примеры применения

Пример 1. В треугольнике ABC чевианы пересекаются в точке O. AF/FB = 2, AE/EC = 3. Найдите AD/OD.

Решение: AD/OD = AF/FB + AE/EC = 2 + 3 = 5.

Пример 2. Для центроида G: AF/FB = 1, AE/EC = 1. Найдите AG/GD.

Решение: AG/GD = 1 + 1 = 2. Известно, что центроид делит медиану в отношении 2:1, считая от вершины, то есть AG:GD = 2:1. AG/GD = 2. Всё верно.

Пример 3. В треугольнике точка O делит чевиану AD в отношении AD/OD = 4. Известно, что AF/FB = 1. Найдите AE/EC.

Решение: AE/EC = AD/OD − AF/FB = 4 − 1 = 3.

Историческая справка

Теорема названа в честь бельгийского математика Эдуарда Ван-Обеля (Edward Van Aubel), который опубликовал её в 1878 году.

Ван-Обель также известен другими теоремами в геометрии, в частности, теоремой о квадратах, построенных на сторонах четырёхугольника.

Теорема Ван-Обеля — одна из классических теорем геометрии треугольника, связывающая отношения отрезков на сторонах с делением чевианы точкой пересечения.

Интересные факты

  • Теорема Ван-Обеля — обобщение свойства центроида (AG:GD = 2:1, сумма отношений = 2).
  • Теорема справедлива для любых трёх чевиан, пересекающихся в одной точке.
  • С помощью теоремы Ван-Обеля можно вычислить, в каком отношении точка пересечения делит каждую из чевиан.
  • Существует также «внешняя» версия теоремы Ван-Обеля для точек на продолжениях сторон.
  • Теорема Ван-Обеля тесно связана с теоремой Чевы и теоремой Менелая.
