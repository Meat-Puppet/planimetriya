Формула Герона

S = √p(p − a)(p − b)(p − c)

Формулировка

Площадь треугольника можно вычислить, зная только длины всех трёх его сторон, по формуле Герона: S = √(p(p − a)(p − b)(p − c)), где p = (a + b + c) / 2 — полупериметр треугольника.

Обозначения

Пусть дан треугольник со сторонами:

a, b, c — длины сторон треугольника.

p = (a + b + c) / 2 — полупериметр треугольника.

S — площадь треугольника.

Тогда: S = √(p(p − a)(p − b)(p − c))

Доказательство

Шаг 1. Площадь треугольника можно выразить через две стороны и синус угла между ними: S = ½·b·c·sin A.
Шаг 2. По теореме косинусов: a² = b² + c² − 2bc·cos A, откуда cos A = (b² + c² − a²) / (2bc).
Шаг 3. Используем основное тригонометрическое тождество: sin²A + cos²A = 1, следовательно, sin A = √(1 − cos²A).
Шаг 4. Подставляем cos A: sin A = √(1 − ((b² + c² − a²) / (2bc))²).
Шаг 5. Приводим к общему знаменателю: sin A = √( (4b²c² − (b² + c² − a²)²) / (4b²c²) ).
Шаг 6. Раскладываем числитель как разность квадратов: 4b²c² − (b² + c² − a²)² = (2bc − (b² + c² − a²)) · (2bc + (b² + c² − a²)).
Шаг 7. Преобразуем: (a² − (b − c)²) · ((b + c)² − a²) = (a − b + c)(a + b − c)(b + c − a)(b + c + a).
Шаг 8. Замечаем, что p = (a + b + c)/2, тогда: a + b + c = 2p; a + b − c = 2(p − c); a − b + c = 2(p − b); −a + b + c = 2(p − a).
Шаг 9. Подставляем в sin A: sin A = √( 16p(p − a)(p − b)(p − c) ) / (2bc) = 2√(p(p − a)(p − b)(p − c)) / (bc).
Шаг 10. Подставляем sin A в формулу площади: S = ½·b·c·sin A = ½·b·c·(2√(p(p − a)(p − b)(p − c)) / (bc)) = √(p(p − a)(p − b)(p − c)).

Формула доказана.

Следствия и частные случаи

  • Прямоугольный треугольник: если c — гипотенуза, то p = (a + b + c)/2, и формула даёт S = ½·a·b.
  • Равносторонний треугольник: a = b = c, тогда p = 3a/2, S = √(3a/2 · a/2 · a/2 · a/2) = a²√3 / 4.
  • Равнобедренный треугольник: b = c, тогда p = (a + 2b)/2, формула упрощается.
  • Формула Герона позволяет вычислить площадь, не зная углов и высот.
  • Для вырожденного треугольника (a + b = c) площадь равна нулю, так как p − c = 0.

Примеры применения

Пример 1. Найдите площадь треугольника со сторонами a = 13, b = 14, c = 15.

Решение: p = (13 + 14 + 15) / 2 = 21. S = √(21 · (21−13) · (21−14) · (21−15)) = √(21 · 8 · 7 · 6) = √(7056) = 84.

Пример 2. Стороны треугольника равны 5, 6 и 7. Найдите его площадь.

Решение: p = (5 + 6 + 7) / 2 = 9. S = √(9 · 4 · 3 · 2) = √(216) = 6√6 ≈ 14,7.

Пример 3. Площадь треугольника равна 24, а его стороны — целые числа, образующие арифметическую прогрессию с разностью 1. Найдите стороны.

Решение: Пусть стороны: a = x − 1, b = x, c = x + 1. p = 3x/2. По формуле Герона: 24 = √( (3x/2)(x/2 + 1)(x/2)(x/2 − 1) ). Решая уравнение, получаем x = 8. Стороны: 7, 8, 9.

Историческая справка

Формула названа в честь древнегреческого математика и инженера Герона Александрийского (около 10–70 гг. н.э.), который привёл её в своей работе «Метрика».

Однако есть свидетельства, что эта формула была известна ещё Архимеду (III век до н.э.), а возможно, и вавилонским математикам.

Формула Герона — одна из самых известных и полезных формул в элементарной геометрии. Она позволяет находить площадь треугольника без вычисления высот и углов.

Интересные факты

  • Существует формула Брахмагупты — обобщение формулы Герона для вписанного четырёхугольника: S = √((p−a)(p−b)(p−c)(p−d)).
  • Формула Герона может быть записана в симметричном виде: 16S² = 2a²b² + 2b²c² + 2c²a² − a⁴ − b⁴ − c⁴.
  • С помощью формулы Герона можно вычислить площадь любого многоугольника, разбив его на треугольники.
  • В древности формула Герона использовалась землемерами для измерения площадей участков неправильной формы.
  • Алгоритм на основе формулы Герона встроен во многие компьютерные программы для геометрических расчётов.
