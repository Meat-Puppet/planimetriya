Сумма углов выпуклого n-угольника

Σ = (n − 2) · 180°

Формулировка

Сумма внутренних углов выпуклого n-угольника равна (n − 2) · 180°, где n — количество сторон (вершин) многоугольника.

Обозначения

Пусть дан выпуклый n-угольник A₁A₂A₃...Aₙ.

n — количество сторон (вершин) многоугольника (n ≥ 3).

Σ — сумма всех внутренних углов многоугольника.

Тогда: Σ = ∠A₁ + ∠A₂ + ... + ∠Aₙ = (n − 2) · 180°

Доказательство

Шаг 1. Выберем произвольную точку O внутри выпуклого n-угольника.
Шаг 2. Соединим точку O со всеми вершинами многоугольника. Получим n треугольников.
Шаг 3. Сумма углов всех n треугольников равна n · 180°.
Шаг 4. Эта сумма включает все внутренние углы многоугольника Σ и все углы при вершине O, которые в сумме составляют полный угол 360°.
Шаг 5. Составим уравнение: n · 180° = Σ + 360°.
Шаг 6. Выразим Σ: Σ = n · 180° − 360° = (n − 2) · 180°.

Теорема доказана.

Альтернативное доказательство (через диагонали)

Шаг 1. Проведём из одной вершины n-угольника все диагонали к несоседним вершинам.
Шаг 2. Эти диагонали разбивают n-угольник на (n − 2) треугольника.
Шаг 3. Сумма углов всех этих треугольников равна сумме внутренних углов n-угольника.
Шаг 4. Каждый треугольник имеет сумму углов 180°, всего треугольников (n − 2).
Шаг 5. Следовательно, Σ = (n − 2) · 180°.

Это доказательство наглядно и не требует введения внутренней точки.

Частные случаи

  • Треугольник (n = 3): Σ = (3 − 2) · 180° = 180°.
  • Четырёхугольник (n = 4): Σ = (4 − 2) · 180° = 360°.
  • Пятиугольник (n = 5): Σ = (5 − 2) · 180° = 540°.
  • Шестиугольник (n = 6): Σ = (6 − 2) · 180° = 720°.
  • Восьмиугольник (n = 8): Σ = (8 − 2) · 180° = 1080°.

Следствия из теоремы

  • Сумма внешних углов любого выпуклого n-угольника (взятых по одному при каждой вершине) равна 360°, независимо от n.
  • Каждый внутренний угол правильного n-угольника равен (n − 2) · 180° / n.
  • Количество диагоналей выпуклого n-угольника равно n(n − 3) / 2.
  • С увеличением n сумма внутренних углов растёт линейно.
  • Формула справедлива для любого выпуклого многоугольника, включая невыпуклые (если учитывать знаки углов).

Примеры применения

Пример 1. Найдите сумму внутренних углов семиугольника.

Решение: Σ = (7 − 2) · 180° = 5 · 180° = 900°.

Пример 2. Сумма внутренних углов многоугольника равна 1260°. Сколько у него сторон?

Решение: (n − 2) · 180° = 1260° ⇒ n − 2 = 7 ⇒ n = 9. Это девятиугольник.

Пример 3. Найдите угол правильного двенадцатиугольника.

Решение: Сумма углов: (12 − 2) · 180° = 1800°. Каждый угол: 1800° / 12 = 150°.

Историческая справка

Формула суммы углов многоугольника была известна ещё древнегреческим математикам. Пифагорейцы (VI век до н.э.) знали, что сумма углов треугольника равна 180°, а четырёхугольника — 360°.

Обобщение на n-угольник приписывается Евклиду, хотя в «Началах» оно явно не формулируется. В современном виде формула появилась в учебниках геометрии Нового времени.

Интересно, что для невыпуклых многоугольников формула также работает, если учитывать знаки углов (положительные для выпуклых, отрицательные для вогнутых).

Интересные факты

  • При n → ∞ сумма внутренних углов стремится к бесконечности, а каждый угол правильного n-угольника стремится к 180°.
  • Формула (n − 2) · 180° работает только в евклидовой геометрии. На сфере сумма углов n-угольника больше, а на гиперболической плоскости — меньше.
  • Паркеты (замощения плоскости) правильными многоугольниками возможны только для n = 3, 4, 6, так как их углы должны делить 360° нацело.
  • В архитектуре и дизайне знание суммы углов помогает рассчитывать повороты при создании многоугольных конструкций.
  • Формула суммы углов — одна из немногих, которые запоминают на всю жизнь даже те, кто давно забыл школьную геометрию.
